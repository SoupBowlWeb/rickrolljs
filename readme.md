# RickRoll.js

RickRoll.js Rick Rolls a user when a given keyword is entered on the web page.

```html

<script src="jquery.js"></script>
<script src="/js/rickroll.js"></script>
<script>
    $().rickroll();
</script>

```

```html

<script src="jquery.js"></script>
<script src="/js/rickroll.js"></script>
<script>
    $().rickroll({
        keyword: 'foobar'
    });
</script>

```
